### Key Finding: Option 3 → Option 5

Option 3 (`importNetworkFromPyTorch`) fails at C code generation because ShuffleNet V2's channel shuffle, chunk, and mean operations generate 30 custom layers inheriting from `nnet.layer.AutogeneratedFromPyTorch` — a base class that MATLAB Coder cannot compile.

**Option 5 rescues this** by replacing all 30 layers with codegen-compatible equivalents:
- 16 `view_view` layers → `ChannelShuffleLayer` (reshape + permute)
- 13 `chunk` layers → `ChannelSplitLayer` (channel indexing)
- 1 `mean` layer → `globalAveragePooling2dLayer` + `flattenLayer`

Numerical equivalence: max error **4.77e-06**, cosine similarity **1.0000000000**.
